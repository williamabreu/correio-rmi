DESCRIÇÃO
=========

Serviço básico de correio eletrônico usando tecnologia Java RMI

Compilado e testado para:
- bash 4.4.19 (linux)
- Java JRE 10.0.2


Execução como servidor:

-------------------------------
$  sh server.sh <host> <port>
-------------------------------


Execução como cliente:

-------------------------------
$  sh client.sh <host> <port>
-------------------------------


Para teste local, usar em ambos (cliente e servidor):
  <host> = localhost
  <port> = 0
